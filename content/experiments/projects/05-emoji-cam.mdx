Created in collaboration with Zichen Yuan, this project was developed for the Introduction to Computational Media class at NYU ITP. The challenge was to build a weekly series of projects utilizing different inputs and interactions through p5.js. This specific experiment takes a live video feed and translates each individual pixel into a corresponding emoji in real-time.

The project was built entirely from scratch with pure code, utilizing no external software supports or pre-made filters. It served as a foundational exercise in managing high-density data arrays and real-time visual processing within a browser environment. Through this iterative coding process and collaboration with peers, I developed the coding logic, vocabulary knowledge and artistic skills that I still apply to many commercial and gallery projects that involve programming.
